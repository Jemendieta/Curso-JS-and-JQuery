de JavaScript

JS tiene 3 objetos prncipales:
         document: que permite acceder y recorrer los elementos html.
         window: que suele controlar ventanas emergentes por ejemplo.
         navigator: hace referencia al navegador y sus propiedades, por ejemplo google analitics.

dos formas básicas de seleccionar elementos en js es usando: 

            document.getElementById()
            document.getElementByTagName()
ambos pertenecientes al objeto document. Pudienso seleccionarlos or su Id o por la propia etiqueta.

de Jquery

el objeto jquery: sintaxys $("selector correspondiente") de esta forma podemos seleccionar un elemento de nuestro html

ejemplo

         //ejemplo con js
            var el_mensaje=document.getElementById("mensaje").value;
         //ejemplo con jquery
            var el_mensaje=$("#mensaje").value;

función ready

permite cargar nuestro código jquery despues de que el DOM esté cargado

Funciones Jquery para agregar contenido

funcion html

sustituye un elemento de nuestro código en este caso una etiqueta html, ademas reemplaza el texto que podamos tener en nuestra web

también tenemos la propiedad innerHTML la cual sustituye el texto de un elemento

funcion append()

agregar un nuevo texto a continuación

funcion prepend

agrega un nuevo texto antes

before y after

funciones muy usadas en css que en jquery tienen igual propósito y funcionamiento

funcion text()

funciona similar a la función html con la diferencia de que no es capaz de manipular etiquetas, solo sustituye texto
